# README
[中文]
在进行学术论文写作时，部分期刊自行定义参考文献的格式，知网提供的有限参考文献格式不全或不支持，手动调整较为麻烦，因此本人打算编写一个参考文献的自动转换工具。

**初步功能包括：** 解析常见通用格式；通用格式之间的互相转化；支持自定义参考文献的格式；支持由通用格式向自定义参考文献格式转换。

工具提供windows平台可视化界面，后续可能提供Linux平台软件。使用python编写。
